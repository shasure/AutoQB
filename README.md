# 项目说明

本项目使用 Python 编写，依赖 `pandas` 和 `playwright` 库来实现特定功能。以下是安装依赖的步骤：

## 安装依赖

### 创建虚拟环境（可选但推荐）
推荐conda

## 安装依赖
使用以下命令安装项目所需的依赖：
```bash
pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple
```

### 安装 Playwright 浏览器驱动
安装 playwright 后，还需要安装浏览器驱动：

```bash
playwright install
```
